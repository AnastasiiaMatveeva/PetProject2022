# parser .cbd files

support keywords: 'ET', 'NBLOCK', 'EBLOCK', 'MPDATA', 'SECOFFSET', 'SECBLOCK' of ANSYS Preprocessor

the folder "input data" contains .cdb file for parsing

launch parser.py in the folder "preprocessor"

# Dependencies

Required: 
        
        regex == 2021.4.4
        pip == 22.0.4
